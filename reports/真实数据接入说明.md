# 真实数据接入说明

当前仓库还没有真实细胞图像与标注数据，因此正式实验还不能直接开始。

已经补好的接入链路：

- 原始数据目录：
  - `data/formal_dataset/raw/images`
  - `data/formal_dataset/raw/masks`
- 自动切分脚本：
  - `research/datasets/prepare_dataset.py`
- 正式配置模板：
  - `configs/formal_dataset_template.yaml`
- 一键 PowerShell 脚本：
  - `scripts/prepare_formal_dataset.ps1`

## 1. 原始数据要求

请把真实数据按“平铺目录”放入：

```text
data/formal_dataset/raw/
  images/
  masks/
```

要求：

- 图像和标注文件名的主干必须一致。
- 例如：`cell_001.png` 对应 `cell_001.png`
- 图像支持：`.png .jpg .jpeg .tif .tiff .bmp`
- 标注支持：`.png .tif .tiff .bmp`

## 2. 自动切分

执行：

```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\prepare_formal_dataset.ps1
```

执行后会生成：

```text
data/formal_dataset/processed/
  train/
    images/
    masks/
  val/
    images/
    masks/
  test/
    images/
    masks/
  manifest.json
```

## 3. 正式训练

切分完成后，可以先复制一份正式配置模板，再开始训练：

```powershell
Copy-Item configs\formal_dataset_template.yaml configs\formal_baseline.yaml
python -m research.engine.train --config configs/formal_baseline.yaml
```

## 4. 后续建议

正式数据接入后，建议按下面顺序继续：

1. 跑 baseline
2. 跑 se
3. 跑 cbam
4. 跑 se_cbam
5. 统一做消融汇总
6. 再做可视化和测速
